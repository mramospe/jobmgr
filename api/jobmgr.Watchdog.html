Watchdog¶

class jobmgr.Watchdog[source]¶

Bases: object

Object to iterate over a set of jobs and update its status. The objects are passed throguh the Watchdog.watch() method. To be thread-safe, one must ensure to do not delete the jobs before stopping the watchdog monitoring (through Watchdog.stop()).

Methods Summary

start() Start monitoring the jobs.
stop() Stop monitoring the jobs.
watch(job) Start monitoring the given job.

Methods Documentation

start()[source]¶

Start monitoring the jobs.

stop()[source]¶

Stop monitoring the jobs.

Note

This must be done before deleting the jobs, to prevent destroying jobs in the queue.
