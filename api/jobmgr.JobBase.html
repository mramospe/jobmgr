JobBase¶

class jobmgr.JobBase(path, kill_event=None, registry=None)[source]¶

Bases: object

Base class to create a directory for a job, holding also an ID and a status code.

Parameters:
  • path (str) – path to the desired directory.
  • registry (JobRegistry or None) – instance to register the object. If “None”, the object will be registered in the main JobManager instance.
Variables:

jid – Job ID, determined by the subdirectories in the output path.

Methods Summary
